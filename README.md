
📱 EasyCar – Aplicação Mobile (React Native)

Descrição:
O EasyCar é uma aplicação mobile desenvolvida em React Native, criada para facilitar o transporte urbano, conectando passageiros e motoristas de forma eficiente. Através de uma interface intuitiva e fluída, os utilizadores podem solicitar corridas, acompanhar o status da viagem e interagir com os motoristas em tempo real.

A aplicação comunica-se com a API do EasyCar para gerir as solicitações de corridas, garantindo uma experiência fluída e confiável.

🔹 Funcionalidades Principais

✅ Para Passageiros:
📍 Solicitação de corridas informando ponto de partida e destino.
🚖 Acompanhamento em tempo real do status da corrida.
❌ Cancelamento de corridas antes da confirmação do motorista.
🔄 Restrição: apenas uma corrida activa por vez.

✅ Para Motoristas:
📥 Recebimento e aceitação de corridas disponíveis.
📌 Exibição do endereço de recolha e destino do passageiro.
❌ Cancelamento de corridas caso necessário.
✅ Finalização da corrida após a chegada ao destino.
🔄 Restrição: apenas uma corrida aceita de cada vez.

✅ Outras Funcionalidades:
🗺️ Integração com Google Maps API para localização e rotas.
📊 Histórico de Corridas para utilizadores acompanharem viagens passadas.
📣 Notificações push para alertas sobre mudanças no status da corrida.

🛠 Tecnologias Utilizadas
Linguagem & Framework: React Native (Expo ou CLI)
Gerenciamento de Estado: Context API / Redux Toolkit
Mapas & Geolocalização: Google Maps API & React Native Maps
Armazenamento: AsyncStorage para cache local e SQLite para histórico de corridas
Comunicação com Backend: Axios para chamadas à API


🎯 Objectivo do Frontend
Proporcionar uma experiência fluída e intuitiva para passageiros e motoristas, garantindo uma interação eficiente com o backend do EasyCar. O foco é criar uma aplicação responsiva, rápida e fácil de usar, permitindo que qualquer pessoa solicite e gerencie corridas com apenas alguns toques.
